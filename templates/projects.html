Projects

  • Facility App is an application I built for tracking maintenance, working orders and assets for facility management. Users can log in and create working orders, print them and track the the maintenance current status.
  • Book's Corner is a book review website I did during the HarvardX course. Once users log in, they are able to search for books, leave reviews for individual books and see the reviews made by other people. I also used a third-party API by Goodreads, another book review website, to pull in ratings from a broader audience. Additionally, users will be able to query for book details and book reviews programmatically via a website's API.
  • Weather channel Project is a weather app that shows the current weather contitions form any country or location. It detect your local IP and returns your location. Then with https://openweathermap.org/ API returns data from any location you search.
  • To do list is a simple web application where you can add multiple tasks on a list, mark them as complete or erase each of them.
